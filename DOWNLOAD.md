# Download Full Dataset

The `camus-warehouse.db` file is included in this repo at `data/camus-warehouse.db`.

For the **full operational database** (2.7 GB, 47 tables, all raw ticks + 245K post-exit samples + 147K trajectory samples), contact Ario for a private download link.

The full dataset is not publicly hosted to avoid excess traffic on the source server, which operates on a tailscale-only network.
